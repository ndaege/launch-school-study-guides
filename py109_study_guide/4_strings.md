Strings are a fundamental data type. They are immutable in Python. Strings can include a variety of ASCII characters, including numbers, as part of the string. For example, the string `"I've eaten 2 apples so far"`. This string is an entire sentence, it even includes a number. 

Strings can also be converted to other data types using their respective constructor function. For example: 

```python
my_str = '100'

my_int = int(my_str)
```

The code above uses the `int()` constructor function to implicitly convert the string `'100'` to the int `100`.
